# Historical Development

Mainly Text conversations - A bit of talking with AI

1950s - 1960s : The start of AI
    1961 : Early speech recognition system, Audrey by Bell Labs and IBM's Shoebox voice-activated calculator

    1966 : Eliza a machine that used methods of recognizing language patterns to mimic human-like conversations

1980s - 2000s : An Era of Text-Based Chatbots
    1988 : Jabberwacky is a "First" Chatbot and its stated to aim to simulate natural human chat in a humourous manner

    1995 : ALICE is a language proccesing chatbot that portrays a young woman who talks about her hobbies and answer your question

    1999 : Clippy suggests random changes in your text input

2010s : The start of AI Voice Assistance era
    2011 : Apple Siri uses voice queries, gesture based control and a natural-language user interface to answer questions and make recommendations. Siri also adapts to the individual user their language usage, searches and preferences to return the best possible result for each user

    2012 : Google Now - Google’s first-ever virtual assistant, Google Now, was the company’s attempt to provide personalized AI assistance. It came with voice search functionality and was integrated with Google Search to offer relevant updates based on user activities

    2014 : Amazon Alexa and Echo revolutionized commercial space but mainly know for revolutionizing residental areas to make smart homes, thanks to the smart voice-activated functionality

    2015 : Microsoft Cortana is a cloud-based personal assistant that operates outside the realm of standard voice-enabled AI. Cortana doesn't just understand voice commands and carry out tasks but is integrated for use across Microsoft's 365 suite of products

2020s : AI Assistants and Generative AI

    2021 : Github AI Copilot AI assistant that helps users write better code. It understands the context behind existing code and suggests new contextually accurate code lines or updates in existing ones

    2022 : ChatGPT is an AI model created by OpenAI that is capable of conversation in text or in voice. The smart tool can answer questions, converse with people, generate content, and generate images. It learns based on your conversations and tries to satisfy you with the best answer possible. Its currently the most used AI model

    2023 : Microsoft 365 Copilot a perfect assistant for comprehensive management across its entire ecosystem. It claims to transform productivity in many ways across multiple Microsoft applications. It will assist users in creative tasks by optimizing PowerPoint presentations and improving analysis and data visualization on Excel sheets and much more
